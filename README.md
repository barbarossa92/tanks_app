# Tanks App

This is a simple tanks fight 2d game written in Go and ReactJs

Just run the following

Backend
```
cd ./tanks_api
go get github.com/gorilla/websocket
go run main.go
```
Frontend

```
cd ./tanks_web
npm i
npm start
```
